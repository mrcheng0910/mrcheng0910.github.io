  • markdown
markdown基本用法

记录所有markdown语法，方便以后查找使用

一、标题

代码

#标题1
##标题2
###标题3
####标题4
#####标题5
######标题6

显示格式

标题1

标题2

标题3

标题4

标题5
标题6

注意
也可以在文字下面添加-和=,表示一级和二级标题，但是使用#更方便

二、块注释

通过在文字开头添加>表示块注释,当>与文字间添加五个blank时候，块注释的文字会变化
代码

>测试文字

显示格式

测试文字

三、斜体

需要设置为斜体的文字两端使用1个*或者_夹起来
代码
*我是斜体文字*
_我是斜体文字_
显示格式

我是斜体文字
我是斜体文字

四、粗体

需要设置为斜体的文字两端使用2个*或者_夹起来
代码
**我是粗体文字**
__我是粗体文字__
显示格式

我是粗体文字
我是粗体文字

五、无序列表

在文字开头使用*、+、-实现无序列表，注意上述三个符号与文字间添加空格，建议一个文档中只用一种无序列表的表示方式
代码

* 无序列表
+ 无序列表
- 无序列表

显示格式

  • 无序列表
  • 无序列表
  • 无序列表

无序列表可以分级，如下表示（二级*前有两个空格)

* 无序列表1
  * 无序列表2

显示格式

  • 无序列表1
    • 无序列表2

六、有序列表

使用数字后面跟句号（文字与句号前有空格）
代码

1. 有序列表1
2. 有序列表2
3. 有序列表3

显示格式

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

七、链接

Markdown中有两种方式实现链接，内联方式和引用方式

  • 内联方式
百度[网址](http://www.baidu.com/)

百度网址

  • 引用方式

请自己百度查询

八、图片

图片的处理方式和链接的处理类似，也包括内联和引用，只介绍内联

  • 内联方式
！[alt text](/path/to/img.jpg "title") 可以本地路径也可以网络路径

九、代码

  • 方法一：简单文字使用一个代码框,不是单引号，是ESC下面的按键
`code`

code

  • 方法二:多行文字，使用tab或四个空格
你好

nihao

以上为常用的命令
另外推荐几款windows和linux下方便书写Markdown语法的工具

  • windows下的Markdownpad比较方便，可以实时预览，简单方便书写
  • linux下的retext也是比较方便的,可以在软件中心直接下载使用
    另外sublime控件也可以写，都是比较方便的，但是个人觉得还是前两个比较方便
